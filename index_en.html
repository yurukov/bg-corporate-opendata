On the website of NRA you can find a list of all firms registered for VAT. This is optional in Bulgaria as explained here. One can register for updates for this list and download it daily. The problem is that each day you need to download data on 350000 firms or 60Mb (13 Mb zipped). If you miss one day, you can't get the file later. There's also and issue with encoding in the original files.

That's why I automated the process. In the links below you'lll find a folder with ZIP archives of historic VAT firm lists. The format of the files is dds_20121023.csv.zip with 20121023 representing the date 23.10.2012 for easy sorting. The day-to-day changes can be found in a separate list - changes_20121023.csv.zip. There you'll see lines that have been deleted (starting with delete) and those that are inserted (starting with insert). This should make it easier to make reports.

It should be noted that some of the lines in the "changes" list are actually edited. For such records you'll find one insert and one delete\ line usually one after the other. There's also a statistics on the number of daily changes. They include "edited" records. These are those insert/delete lines, that have the same unique firm id (BULSTAT). The changed records do not reflect reregistered companies having a new BULSTAT number.

One can notice also some annomalies. There are 1950 firms with duplicate BULSTAT numbers. 1894 of them with 123456789.
